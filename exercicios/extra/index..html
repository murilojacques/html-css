<h1>
serve para adicionar um titulo


<p>
linha(paragrafo) pode escrever qualquer coisa contanto q esteja entre o par de tags <p> e </p>


<hr>
serve para aicionar uma linha horizontal q passa por um determinado ponto do site


<br>
serve para dar uma quebra de linha (break row) basicamente faz a escrita ir para a linha de baixo inves é um "espaço" tipo
isso
vale pontuar q o espaçamento entre as linha normalmente é feitopor meio de CSS então não se deve utilizar muitos <br> seguidos
